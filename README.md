- Множественные курсоры
- Folding блоков кода
- Миникарта (minimap)
- Вкладки файлов

## 🚀 Быстрый старт

### Вариант 1: Открыть файл
Просто откройте `javier.html` в браузере — никаких установок не требуется!

### Вариант 2: Локальный сервер (рекомендуется)
```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .

# PHP
php -S localhost:8000
```

Затем откройте `http://localhost:8000/javier.html`

## 📖 Использование

### 1. Регистрация/Вход
- Создайте аккаунт или войдите как гость
- Гостевой режим сохраняет данные в localStorage

### 2. Настройка AI
1. Нажмите **🤖 AI** в шапке
2. Выберите провайдера (рекомендуем **DeepSeek** или **Groq**)
3. Вставьте API-ключ
4. Выберите модель
5. Нажмите **💾 Сохранить**

**Где получить ключи:**
- **DeepSeek**: https://platform.deepseek.com/ (бесплатные кредиты)
- **Groq**: https://console.groq.com/keys (бесплатно)
- **OpenAI**: https://platform.openai.com/api-keys (нужна карта)
- **Claude**: https://console.anthropic.com/ (нужна карта)
- **Gemini**: https://aistudio.google.com/apikey (бесплатный лимит)

### 3. Создание проекта
1. Нажмите **📁 Проекты** → **+ Новый**
2. Введите название
3. Начните писать код!

### 4. Работа с файлами
- **Новый файл**: кнопка **+** в панели файлов
- **Переключение**: клик по вкладке или файлу
- **Закрытие**: крестик на вкладке (Ctrl+W)
- **Сохранение**: Ctrl+S (или auto-save)

### 5. Коллаборация
1. Нажмите **👥 Коллаборация**
2. Введите email коллеги
3. Отправьте приглашение
4. Коллега принимает приглашение
5. Редактируйте код вместе!

### 6. Экспорт/Импорт
- **Экспорт ZIP**: меню → 📦 Экспорт
- **Импорт ZIP**: меню → 📥 Импорт
- **Шаринг URL**: меню → 🔗 Поделиться (создаст ссылку)

## 🛠 Технологии

- **Frontend**: Vanilla JavaScript (ES6+)
- **Редактор**: Monaco Editor 0.45.0
- **Сжатие**: Pako (zlib port)
- **Архивация**: JSZip + FileSaver.js
- **Хранение**: localStorage
- **Синхронизация**: BroadcastChannel API
- **Криптография**: Web Crypto API

### Горячие клавиши

| Клавиши | Действие |
|---------|----------|
| `Ctrl+S` | Сохранить |
| `Ctrl+Enter` | Запустить код |
| `Ctrl+W` | Закрыть вкладку |
| `Ctrl+N` | Новый файл |
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` | Повторить |
| `Tab` | Отступ (или принять AI-подсказку) |
| `Shift+Tab` | Убрать отступ |

### Настройки

- **Auto-save**: автосохранение каждые 500мс
- **AI Inline**: автоматические подсказки от AI
- **Тема**: тёмная/светлая
- **GitHub Token**: для публикации gist

## ⚠️ Известные ограничения

1. **CORS**: Claude и Gemini могут блокироваться браузером. Используйте DeepSeek или Groq.
2. **localStorage**: лимит ~5-10MB в зависимости от браузера
3. **BroadcastChannel**: работает только в одном браузере (не кросс-браузерная коллаборация)
4. **Мобильные**: Monaco Editor может работать медленнее на мобильных устройствах

## 🔐 Безопасность

- Все данные хранятся **локально** в браузере
- API-ключи **не отправляются** на сервер Javier
- Прямые запросы к API провайдеров (DeepSeek, OpenAI и т.д.)
- GitHub токены хранятся в localStorage

**Рекомендации:**
- Не публикуйте API-ключи
- Используйте Personal Access Token с минимальными правами
- Регулярно экспортируйте проекты
